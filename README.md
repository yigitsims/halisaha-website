# ⚽ Halı Saha Website

A modern football field booking platform that allows users to register, log in, view holographic field previews, and make online reservations with secure payment integration.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 🏟️ Holographic Field Preview System
- 💳 Secure Payment System
- 📅 Online Reservation System
- 📱 Responsive Design (Mobile & Desktop)
- 🎯 Clean and user-friendly interface

---

## 🧠 Project Purpose

This project was developed to simulate a real-world football field booking system with modern UI concepts and basic full-stack logic using frontend technologies.
